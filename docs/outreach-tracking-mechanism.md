# Tracking Mechanism: AgentNami Outreach Campaign

## 1. Metrics to Track
- **Outreach Volume:** Number of LinkedIn connections/messages and emails sent per platform.
- **Engagement Rate:**
    - LinkedIn: Acceptance rate, response rate, click-through rate on shared content.
    - Email: Open rate, reply rate, bounce rate.
- **Conversion:** Number of demos requested and completed.

## 2. Tracking Tool (Mock-up)
*Use a Google Sheet to track the following columns:*
- Platform (Salesforce/HubSpot/Zoho/NetSuite)
- Prospect Name
- Outreach Channel
- Date Sent
- Status (Sent/Replied/Interested/Demo Scheduled/Closed)
- Notes (Specific feedback/pain points identified)
