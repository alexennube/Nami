# Getting Started with AgentNami

This page provides a quick start for using AgentNami within your CRM automation workflows.

## Prerequisites
- Node.js and npm installed
- Access to your CRM platform integration

## Installation
1. Install dependencies
   ```bash
   npm install
   ```
2. Start the local server or run your integration script as documented in your project

## Quick Start
- Create a simple workflow that fetches contacts and updates notes
- Run the workflow to see autonomous agents in action

## Core Concepts
- Agents: autonomous workers
- Tasks: units of work
- Workflows: sequences of tasks
